# Task 1: Exploratory Data Analysis (EDA) on Iris Dataset

## Objective

Explore and visualize the Iris dataset using Python, Pandas, Matplotlib, and Seaborn.

## Approach

* Loaded and inspected the dataset
* Performed data quality checks
* Generated descriptive statistics
* Created visualizations (Scatter Plot, Histogram, Box Plot, Heatmap)

## Key Insights

* No missing values found
* Strong correlation between petal features
* Setosa species is clearly distinguishable
* Dataset is suitable for classification tasks

## Tools Used

Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook
