# Task 2: Credit Risk Prediction

## Objective

Build a machine learning model to predict loan approval status based on applicant information and financial factors.

## Approach

* Loaded and explored the dataset
* Handled missing values
* Performed data preprocessing and encoding
* Visualized key patterns and relationships
* Trained and evaluated a machine learning model

## Key Insights

* Credit history is one of the most important factors affecting loan approval.
* Applicant income and loan amount influence approval decisions.
* Data preprocessing improves model performance.
* The model can assist in identifying potential credit risk.

## Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook
